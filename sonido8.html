Activity 1 Tutorial

In this simple HTML tutorial, we're creating a basic webpage that shows a short autobiography. We start with !DOCTYPE html to tell the browser we're using HTML5. Then we open the html tag to begin our document. Inside it, we use a head section with a title tag to name our page 'act1HTMLdoc', which shows up on the browser tab. Next, we go to the body tag, where we put all the visible content. We use p tags to create paragraphs for each detail—like name, age, address, sex, course, and school. This helps organize the text neatly on the page. Finally, we close the body and html tags to end our webpage.
